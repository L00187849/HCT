# Hybrid Cloud Technology (HCT)

---

## Description

This repository contains coursework and project documentation for the **Hybrid Cloud Technology** module as part of the **PG Dip in Cloud Technologies** at **ATU Letterkenny**.

The repository is based on the **Electric-Petrol.ie Infrastructure Design** project. It includes server hardware planning, services infrastructure design, and a branch cluster upgrade design. The work demonstrates hybrid cloud infrastructure planning using Microsoft Hyper-V, Windows Server, Active Directory, DNS, DHCP, NTP, backup planning, business continuity, network segmentation, and management implementation.

## Key Topics Covered

- Hybrid cloud infrastructure design
- Microsoft Hyper-V virtualisation
- Windows Server 2022 Datacenter
- Dell PowerEdge server design
- Hyper-V Failover Clustering
- Clustered shared storage
- Active Directory Domain Services
- DNS, DHCP and NTP services
- VLAN and network segmentation
- Backup, disaster recovery and business continuity
- Infrastructure monitoring and management
- Dell iDRAC / BMC remote management
- AAA design: Authentication, Authorisation and Accounting
- Management implementation planning
